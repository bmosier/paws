Pet Adoption Web Server (PAWS)

Designed and Developed By: Samuel James, Ben Mosier and Denis Roman

The Pet Adoption Web Server was created to allow animal shelters, animal foster homes, volunteers and customers to connect through a simple website where customers can easily browse and find nearby shelters or fosterhomes to adopt pets.
